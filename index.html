Git команди

  • git config --global user.email(Встановлює поштову скриньку користувача)
  • git config --global user.name(Встановлює ім’я користувача)
  • git commit -m(Підтвердити зміни)
  • git push(Завантажити зміни на Github)
  • git checkout homework(Переключитись на вказану гілку)
  • git branch homework(Створити нову гілку)
